條件運算式 Conditional 'If'


“If”條件语句是用于检查布尔值（True/False）及根据布尔值控制程序的运作。

一个基本的“If”條件语句可以表示为如下：

if ( condition )
{
   execute this statement if the condition is true
}


该條件语句可以进一步表示为：

if ( condition )
{
   execute this statement if the condition is true
}
else
{
   execute this statement if the condition is false
}



条件否定可使用“！”（惊叹号）

如果惊叹号被放在条件前，该语句的布尔值将被否定。

if ( !condition )
{
   execute this statement if the condition is false
}



OR布尔运算符

OR布尔运算符（||）可以用来连接两个或多个语句，而如果一个或多个的语句是正确的，联合声明只会返回True。

if ( condition1 || condition2 )
{
   execute this statement if condition1 is true or condition2 is true or both are true
}



AND布尔运算符

AND布尔运算符（&&），可用于连接两个或多个语句，而如果所有的语句是正确的，联合声明只会返回True。

if ( condition1 && condition2 )
{
   execute this statement if both condition1 and condition2 are true
}
