变量 Variables


变量是用来存储将在整个程序执行时改变的数值。

 int 型是用来存储整数，例如： 3

double 双类型是被用于存储浮点值，它也可以存储整数太例如： 3.1415, 3.0

char 型是用来存储单个字符，例如： 'a'

boolean 布尔类型是用于存储 真(true)/假(false) 值，例如： true


一个独特的变量名称会被分配一个变量类型，以作为标识符

int a = 5;
